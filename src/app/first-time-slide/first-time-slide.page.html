¡Bienvenido!

¡Gracias por descargar nuestra aplicación de movil!

Con esta aplicación podrás crear geolocalizaciones, ver el mapa en el que colaboran los usuarios, y estar al día sobre las últimas noticias sobre nosotros.

¿En qué consiste el proyecto?

Este proyecto pretende dar visibilidad a un problema común en muchos lugares del planeta, "la basura".

¿Como puedo contribuir?

¡Puedes utilizar está aplicación para mandarnos información sobre ubicaciones donde encuentres residuos abandonados!

¿Como funciona la aplicación?

Pulsa sobre los huecos de arriba con el símbolo "" para sacar una foto. Tienes un límite de hasta 4 fotos por geolocalización

Se te pedira la ubicación del dispositivo móvil para saber donde se encuentra la información

Cuando hayas terminado de sacar las fotos, podrás incluir una pequeña descripción.

Cuando estés satisfecho con las fotos, pulsa sobre botón "Enviar Geolocalización" para enviar los datos que acabas de reunir
